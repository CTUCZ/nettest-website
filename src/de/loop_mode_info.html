Der Wiederholungsmodus erlaubt es, den RTR-Netztest automatisiert wiederholt auszuführen.

Der erste Test wird unmittelbar nach dem Start des Wiederholungsmodus gestartet. Weitere Tests werden gestartet, sobald die Wartezeit (Voreinstellung 30 Minuten) abgelaufen ist. Diese Werte können unter „Einstellungen“ verändert werden. Tests werden durchgeführt, bis die eingestellte Testanzahl (Voreinstellung 10 Tests) erreicht ist. Der Wiederholungsmodus wird nach 2 Tagen automatisch beendet. Die Tests können jederzeit gestoppt werden.

Es wird ausdrücklich darauf hingewiesen, dass aufgrund des Wiederholungsmodus ein Rückschluss auf die testende Person nicht ausgeschlossen ist.
